# Domain: skills

## Purpose
Houses knowledge about how ByteRover skills are categorized and reorganized over time to support discovery and planning.

## Scope
Included in this domain:
- Legacy skill categories and their membership
- Planned future categories and subcategories for skills (knowledge, dev, think, spec)
- Move mappings showing where each skill should migrate

Excluded from this domain:
- Implementation details or operation guides for individual skills

## Ownership
Knowledge Engineering team

## Usage
Consult when referencing the canonical skill taxonomy or planning skill placement changes.
