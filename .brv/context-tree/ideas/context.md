# Domain: ideas

## Purpose
Captures proposals for reorganizing ByteRover skills so future lists match the envisioned structure and formatting.

## Scope
Included in this domain:
- Current skill bucket listings (meta, pa-sdlc, specs, utils)
- Planned future categories (knowledge, dev, think, spec with sub-buckets)
- Move tables establishing where each skill should live after the reorganization

Excluded from this domain:
- Operational execution notes
- Standalone feature discussions unrelated to skill taxonomy

## Ownership
ByteRover context engineering

## Usage
Reference when reviewing or iterating on proposed reorganizations of the 62 skills.
