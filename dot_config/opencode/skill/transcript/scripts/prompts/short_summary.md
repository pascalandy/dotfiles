prompt-title: Short summary creator
---

## Goal

Quick high-level summary to help decide if this video is worth watching in full.

## Task

**STEP 1:** Create a short and sweet summary:

- 2-3 sentence overview of what the video covers
- 5-7 bullet points of the main topics/ideas
- Who this video is best suited for

**STEP 2:** At the end, include a section titled '### Links' with up to three strong keywords for Obsidian linking. Use double bracket notation (e.g., [[expression]]). Always include [[AI Summary]].

## Format Example

```md
## Video Title

### Overview

Brief 2-3 sentence summary of what this video covers and its main thesis.

### Key Points

- Main point 1
- Main point 2
- Main point 3
- Main point 4
- Main point 5

### Who Should Watch

This video is ideal for [target audience] who want to [goal].

### Links

- [[keyword]]
- [[concept]]
- [[AI Summary]]
```

## Rules

- Keep under 200 words total
- Skip everything related to sponsors
- No quotes needed for this format
- Start with H2 title
- Use H3 for sections
- Use `-` for bullet points
- Do not output in a code block, just plain markdown
