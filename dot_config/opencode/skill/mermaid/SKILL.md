# Mermaid

## Règles Générales

- Quote every labels : Une erreur classique : Mermaid interprets them as node shape syntax. You need to escape or quote the label.
- **Ne jamais ajouter d'émoji** dans les diagrammes (compatibilité et professionnalisme)
- Utiliser des identifiants courts et descriptifs
- Préférer les couleurs de la palette Catppuccin Frappé (voir skill `color-palette-frappe`)
- Garder les diagrammes simples et focalisés
