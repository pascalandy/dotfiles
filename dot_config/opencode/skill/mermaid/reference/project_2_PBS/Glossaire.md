## Glossaire

| Terme        | Définition                                                                                        |
| ------------ | ------------------------------------------------------------------------------------------------- |
| **PKI**      | Public Key Infrastructure - Infrastructure à clés publiques                                       |
| **CA**       | Certificate Authority - Autorité de certification                                                 |
| **CRL**      | Certificate Revocation List - Liste de révocation des certificats                                 |
| **OCSP**     | Online Certificate Status Protocol - Protocole de vérification en ligne du statut des certificats |
| **ACME**     | Automatic Certificate Management Environment - Protocole d'automatisation des certificats         |
| **Failover** | Basculement vers un système de secours                                                            |
| **SLA**      | Service Level Agreement - Accord de niveau de service                                             |
