# Frontend Aesthetics Guidelines

## Typography

- Choisir des fonts belles, uniques, intéressantes
- Éviter: Arial, Inter, Roboto, system fonts
- Préférer: fonts distinctives qui élèvent l'esthétique
- Pairer: display font distinctive + body font raffinée

## Color & Theme

- Engager une esthétique cohésive
- Utiliser CSS variables pour consistance
- Couleurs dominantes + accents vifs > palettes timides/équilibrées

## Motion & Animation

- Animations pour effets et micro-interactions
- Prioriser CSS-only pour HTML
- Utiliser Motion library pour React
- Focus: page load orchestré avec staggered reveals (animation-delay)
- Scroll-triggering et hover states surprenants

## Spatial Composition

- Layouts inattendus
- Asymétrie, overlap, flux diagonal
- Éléments qui brisent la grille
- Negative space généreux OU densité contrôlée

## Backgrounds & Visual Details

- Créer atmosphère et profondeur (pas de couleurs solides par défaut)
- Effets contextuels selon l'esthétique:
  - Gradient meshes
  - Noise textures
  - Geometric patterns
  - Layered transparencies
  - Dramatic shadows
  - Decorative borders
  - Custom cursors
  - Grain overlays

## Anti-patterns (NEVER)

| Éviter                    | Pourquoi            |
| ------------------------- | ------------------- |
| Inter, Roboto, Arial      | Fonts génériques AI |
| Purple gradients on white | Cliché AI           |
| Layouts prévisibles       | Cookie-cutter       |
| Space Grotesk partout     | Convergence AI      |

## Tone Options

Choisir un extrême et l'exécuter avec précision:

- Brutally minimal
- Maximalist chaos
- Retro-futuristic
- Organic/natural
- Luxury/refined
- Playful/toy-like
- Editorial/magazine
- Brutalist/raw
- Art deco/geometric
- Soft/pastel
- Industrial/utilitarian
