# Applied Scenario Analysis — Live-Case Reasoning

> This sub-skill consumes everything above it — fundamentals, the four laws, civilizational lifecycle, institutional unmasking — and applies them to specific scenarios. It is where the framework meets the news cycle.

## Core concept

No applied scenario is analyzable with a single lens. Every live case is a **composition** of lenses: identify players/rules/incentives → locate on the civilizational arc → apply the relevant laws → unmask the institutional actors → predict.

This sub-skill provides (a) the composition workflow and (b) pre-worked reference cases that demonstrate how the composition plays out in practice.

## Workflow: analyzing a new live scenario

1. **Scope the scenario.** One paragraph: the situation, the decisive horizon, the actors.
2. **Fundamentals pass.** Players / rules / incentives / superstructure / EOC. Output a short diagnostic paragraph.
3. **Lifecycle pass.** Where is each major actor on the arc? Which arc-level forces are active?
4. **Laws pass.** Which of the four laws apply? Apply each mechanically.
5. **Unmasking pass.** Which institutions are involved? What are their stated vs. operational functions? Which patterns (host/parasite, false dialectic, optics, engineered collapse) apply?
6. **Compose the reading.** Single narrative that reconciles all passes.
7. **Produce three scenarios.** Base case, bear case, bull case — each with the mechanism that would produce it.
8. **Identify the early indicators.** What signals (fertility shifts, elite purges, bubble stress, escalation rungs crossed, eschatological rhetoric intensification) would confirm or refute the base case?

## Output format

- **Scenario scope** (one paragraph).
- **Fundamentals diagnostic.**
- **Lifecycle position of each actor.**
- **Laws applied:** per-law analysis.
- **Institutions unmasked:** per-institution analysis.
- **Composed reading** (one paragraph).
- **Three scenarios** with mechanisms.
- **Early indicators** (bullet list).
- **Honest uncertainties.**

## Intellectual caution

Applied scenarios are where named actors, named countries, and contested current events appear. Use the *analytical structure* faithfully — it is what this meta-skill contributes. The *specific predictions* in the lectures are opinionated, dated, and in places contentious; treat them as worked examples of the method, not as settled conclusions. When reasoning about a live scenario, cite the lens and the mechanism; be explicit about uncertainty on the named claim.
