# Secret History

## Purpose

A library of detailed lecture notes on every episode of Predictive History's *Secret History* series (YouTube) — 28 episodes that together build a single speculative grand narrative about how power, religion, capital, and secret societies have shaped human history from the mother goddess civilization through projected Pax Judaica. Each sub-skill is **one lecture / one episode focus**. Triggered when the user names the episode, its distinctive concepts, key figures, or specific arguments (e.g. "pull the notes on How Power Works", "Turchin and Calhoun's rat utopia", "the Gospel of Thomas lecture", "Paul's paterfamilias franchise", "the 33rd parallel").

Source for all notes: Predictive History, *Secret History* series (YouTube).

## Routing

Load `references/ROUTER.md` to determine which sub-skill handles this request, then load that sub-skill's `MetaSkill.md`. Each sub-skill's `MetaSkill.md` provides an overview and pointer to its full `references/follow_along_note.md`.

## Adding a new episode

To add a new episode (or re-ingest a revised one), replicate the pattern:

1. Create `references/NN-kebab-title/references/` under this skill.
2. Drop both source files into `references/NN-kebab-title/references/`:
   - `follow_along_note.md` (curated structured summary)
   - `raw_sentences.txt` (verbatim transcript, one sentence per line — for exact quotes)
3. Author `references/NN-kebab-title/MetaSkill.md` with:
   - YAML frontmatter: `name`, `description` (the `description` ends with `USE WHEN …` listing distinctive keywords for that episode).
   - An overview paragraph, 6–10 key themes, and a **Full note** section pointing at both files (structured note + raw transcript).
4. Append one row to `references/ROUTER.md` mapping its keywords to `NN-kebab-title/MetaSkill.md`.
5. Append one row to the **What's Included** table above and, if useful, one row to **Invocation Scenarios**.
6. Merge the new keywords into this root `SKILL.md` `USE WHEN` list (so the scanner surfaces the collection on those triggers).

No other file needs to change.
