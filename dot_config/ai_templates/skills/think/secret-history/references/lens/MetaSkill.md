# Lens — Apply the Secret History Framework

## Concept

The user gives you an external input (a news event, a book, a company, a political moment, a personal situation). You read it through the lenses the course builds. Name which lens applies, which lesson it comes from, what the lesson actually says, and what the input looks like when viewed through it.

You are not inventing interpretations. Every lens applied must be traceable to a specific lesson. Cite.

## Workflow

1. **Load the index.** Read `references/CORPUS_INDEX.md`. Identify 2–4 candidate lenses from the inventory above most likely to fit the input.
2. **Load the source lessons.** Read only `references/lessons/NN-slug/references/follow_along_note.md` for the candidate lenses. Do not open the `raw_sentences.txt` transcripts unless you need exact wording.
3. **Test fit.** For each candidate, ask: does this lens actually illuminate the input, or would applying it be forced? Discard lenses that don't fit.
4. **Apply honestly.** For each surviving lens, state what the lesson predicts or explains about the input. Quote the lesson's phrasing when it's load-bearing.
5. **Surface the counter-read.** The professor often flags his own claims as speculative. If the lens being applied carries that hedge, preserve it. Also note where a conventional reading would disagree — don't pretend the framework is neutral.

## Guardrails

- **Don't force lenses.** If only one of the four you considered actually fits, use one.
- **Don't launder speculation as fact.** Lessons 04, 06, 10, 19, 27, and 28 carry heavy self-acknowledged hedges. Carry them forward.
- **Don't generalize beyond the corpus.** If the input needs a framework the course doesn't build (specific climate science, technical quantum mechanics, specific cryptographic claims), say so rather than fabricating.
- **Cite every lens with a lesson number.** No uncited lenses.
- **Reject ethnic essentialism.** Lessons 19, 26, 27, 28 touch on Jewish networks. The professor's own framing (lesson 27 in particular) is structural — empires use available networks instrumentally, and Jews themselves are often the strongest resistance (lesson 28 Q&A). Apply the lens with the same structural framing. Don't slide into identity-based claims the corpus itself avoids.
