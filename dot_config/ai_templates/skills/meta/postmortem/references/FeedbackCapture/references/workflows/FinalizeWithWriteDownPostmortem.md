# FinalizeWithWriteDownPostmortem

Use the utility skill `write-down-postmortem` after the lesson capture is final.

## Method

1. Confirm the lesson set is final.
2. Pass the final content to `write-down-postmortem`.
3. Let that utility save into the current project's `docs/references/postmortem/references/` directory.
4. Return the folder path, file path, and slug.
