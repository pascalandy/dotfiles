# CaptureFromCompletedWork

Use after a completed task, project slice, or working session.

## Method

1. State the context in 1-2 lines.
2. Extract the main lessons.
3. Separate what should be repeated from what should be avoided.
4. Add agent and user feedback only if it changes future execution.

## Output

- Context
- Lessons
- What To Repeat
- What To Avoid
- Feedback for the Agent
- Feedback for the User
