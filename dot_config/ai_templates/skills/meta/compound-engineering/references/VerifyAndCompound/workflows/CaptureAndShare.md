# CaptureAndShare

Use this workflow when the change is done and the remaining work is to preserve, present, or polish the output.

## Route Selection

| Need | Load |
|---|---|
| Document a solved problem as a reusable learning | `../../../skills/ce-compound/SKILL.md` |
| Refresh stale learning docs or pattern guidance | `../../../skills/ce-compound-refresh/SKILL.md` |
| Generate a changelog from merged work | `../../../skills/changelog/SKILL.md` |
| Record a walkthrough video for the feature or PR | `../../../skills/feature-video/SKILL.md` |
| Polish prose to match Every's editorial style | `../../../skills/every-style-editor/SKILL.md` |

## Expected Result

- A durable record of the change and why it matters
- Better reviewer and future-maintainer ergonomics
- Knowledge that compounds instead of evaporating after the merge
