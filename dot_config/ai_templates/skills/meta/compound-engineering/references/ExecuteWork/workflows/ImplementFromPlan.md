# ImplementFromPlan

Use this workflow when the request is to execute a prompt, a plan, a spec, or a todo list.

## Route Selection

| Need | Load |
|---|---|
| Standard implementation from a plan, prompt, or todo | `../../../skills/ce-work/SKILL.md` |
| Implementation that explicitly calls for the beta delegate path | `../../../skills/ce-work-beta/SKILL.md` |
| Full autonomous execution of a larger engineering workflow | `../../../skills/lfg/SKILL.md` |
| Full autonomous execution with swarm-style parallelism | `../../../skills/slfg/SKILL.md` |
| Need explicit swarm orchestration guidance before or during execution | `../../../skills/orchestrating-swarms/SKILL.md` |

## Expected Result

- Working code or completed implementation artifacts
- Tests or verification run as part of execution when appropriate
- A ready handoff into delivery or review
