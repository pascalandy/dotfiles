# Discover Sources

Use this workflow when the user needs better public-source coverage for a niche investigation area.

## Steps

1. Define the gap.
State the investigation domain, geography, language, entity type, or time period where current source coverage is weak.

2. Gather candidate sources.
Search for public directories, official registries, research communities, maintained tool lists, and credible practitioner references relevant to that niche.

3. Evaluate each candidate.
Only keep sources that are relevant, still maintained, publicly reachable, and meaningfully different from the sources already known.

4. Organize the shortlist.
Group sources by purpose such as registries, search aggregators, social discovery, domain intelligence, threat intelligence, archives, or public records.

5. Recommend additions.
Explain what each source is useful for, its limitations, and why it deserves a place in the investigation toolkit.

## Deliverable

Return a shortlist of new sources with purpose, trust level, limitations, and the investigation step each one improves.
