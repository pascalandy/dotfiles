# Design Direction

Use this workflow when the request is about visual direction, design systems, design exploration, or selecting a UI direction before code generation or polish work.

| Request Pattern | Load This Skill | Why |
|---|---|---|
| design system, brand, visual language, foundational UI direction | `../../design-consultation/SKILL.md` | Establish the design system and overall direction |
| generate multiple design variants, compare options, pick a direction | `../../design-shotgun/SKILL.md` | Explore alternatives before committing |
| review the design of a plan before implementation | `../../plan-design-review/SKILL.md` | Critique the planned UX before building |

Expected artifact: a selected design direction, design system, or reviewed design plan.
