# Risk And Performance

Use this workflow when the request is about security review, threat modeling, performance baselining, or evidence that the system can hold up under real use.

| Request Pattern | Load This Skill | Why |
|---|---|---|
| security review, threat model, OWASP, exploit risk, auth exposure | `../../cso/SKILL.md` | Run the security specialist |
| benchmark, performance baseline, compare before and after, web vitals | `../../benchmark/SKILL.md` | Measure performance and compare changes |

Expected artifact: verified risks with exploit scenarios, or benchmark evidence suitable for release decisions.
