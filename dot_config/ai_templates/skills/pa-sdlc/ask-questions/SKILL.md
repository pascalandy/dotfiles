## Q&A format with intelligent options

When you need clarification, interview the user to resolve ambiguity and make the work tighter.

**RULES**:
- Ask up to three questions at a time
- Provide answer choices like: `a) ... b) ... c) ...` so the user can reply quickly
- Show your recommended option first
- Sort choices by highest probability first

**IMPORTANT**: Each answer the user provides must inform the next question(s) you ask. Limit yourself to three questions at a time, so be strategic.

When appropriate, continue with another round of questions
