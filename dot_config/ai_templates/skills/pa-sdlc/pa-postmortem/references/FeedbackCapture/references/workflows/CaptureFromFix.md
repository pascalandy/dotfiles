# CaptureFromFix

Use when the source material is a bug fix, repair, or debugging outcome.

## Method

1. State the issue and the fix.
2. Identify what allowed the issue to happen.
3. Extract the durable lesson from the fix.
4. Capture guardrails or checks that should exist next time.

## Output

- Context
- Issue
- Fix
- Lessons
- Future Guardrails
