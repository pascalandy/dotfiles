# QuickPostmortem

Use for smaller incidents where a full report would be disproportionate.

## Method

1. Summarize the failure and impact in 2-4 lines.
2. Capture the most likely supported cause.
3. Note the fix.
4. List 1-3 follow-ups if any remain.

## Output

- Summary
- Impact
- Cause
- Fix
- Follow-up
