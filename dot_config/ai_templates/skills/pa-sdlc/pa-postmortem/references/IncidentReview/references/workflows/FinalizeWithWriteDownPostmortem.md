# FinalizeWithWriteDownPostmortem

Use the utility skill `write-down-postmortem` after the incident review is final.

## Method

1. Confirm the incident postmortem is complete.
2. Pass the final content to `write-down-postmortem`.
3. Let the utility save it inside the current project's `docs/references/postmortem/references/` directory.
4. Return the folder path, file path, and slug.
