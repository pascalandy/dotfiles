# ExtractLessonsFixesFeedback

Turn the classified memories into durable takeaways.

## Required Questions

For the session as a whole, extract:

- What did we learn?
- What went wrong?
- What did we discover?
- What was the fix?
- What feedback should the agent keep?
- What feedback should the user keep?

## Method

1. Start from the classified memory list.
2. Merge overlapping items instead of repeating them.
3. Keep lessons actionable and specific.
4. Distinguish corrective actions from observations.
5. Preserve the user's own process and language when that matters.

## Output

Produce the sections:

- `What Went Wrong`
- `What We Discovered`
- `Fixes Applied`
- `Feedback for the Agent`
- `Feedback for the User`
