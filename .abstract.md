# Abstract

This is a personal `chezmoi` source repo for home-directory configuration, shell and Git setup, local scripts, AI agent templates, and apply-time automation. Use it when you need the canonical source file behind a managed dotfile, want to trace what `chezmoi apply` syncs automatically, or need to find where agent commands, skills, and app-specific configs actually live before reading the source files directly.

## Use this when
- You need the source for a managed shell, Git, editor, or CLI config.
- You need to trace behavior that happens before or after `chezmoi apply`.
- You need the source of truth for agent prompts or skills under `dot_config/ai_templates/`.
- You need to find custom user scripts in `dot_local/bin/`.

## Main areas
- `dot_zshrc` — Main interactive shell behavior, aliases, and tool setup.
- `dot_config/` — App configs, AI tooling, editor settings, and service-specific files.
- `.chezmoiscripts/` — Automation that syncs external machine state back into this repo.
- `dot_local/bin/` — User-maintained executable scripts exposed via `~/.local/bin/`.
- `./.overview.md#reference-map` — Detailed map

## Out of scope
- Sensitive or machine-specific material under `private_*`; inspect only when the task explicitly requires it.
- Generated, ignored, or copied state such as caches, package directories, and apply-time outputs.
- Exact runtime behavior when the source alone is ambiguous; use L2 files and `chezmoi` commands.
