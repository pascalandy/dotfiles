# Abstract

Personal `chezmoi` source repo for dotfiles, shell/Git setup, local scripts, AI agent templates, and apply-time automation. Use it when you need the canonical source behind a managed dotfile, want to trace what `chezmoi apply` syncs, or need to find where agent commands/skills live.

## Use this when
- You need source for a managed shell, Git, editor, or CLI config.
- You need to trace behavior before/after `chezmoi apply`.
- You need the source of truth for agent prompts or skills under `dot_config/ai_templates/`.
- You need custom user scripts in `dot_local/bin/`.

## Main areas
- `AGENTS.md` — Entry point for AI agents with repo conventions.
- `dot_zshrc` — Main interactive shell behavior, aliases, tool setup.
- `dot_config/` — App configs, AI tooling, editor settings.
- `dot_config/ai_templates/` — Active agent commands and skills.
- `.chezmoiscripts/` — Automation that syncs external state back.
- `dot_local/bin/` — User-maintained executable scripts.

## Out of scope
- Sensitive material under `private_*`; inspect only when explicitly required.
- Archived areas: `bienvenue_chez_moi/`, `*_archives/`, `*_archived/`.
- Generated/ignored state: caches, package directories, apply-time outputs.
